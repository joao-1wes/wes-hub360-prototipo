Criar bot no BotFather

Passo a passo para gerar o token e voltar para a integração.

Como gerar o token do Telegram

Siga essa sequência no Telegram e depois volte para colar o token na etapa 1.

Abrir @BotFather open_in_new
  1. No Telegram, abra o @BotFather e toque em Start ou envie /start.
  2. Envie o comando /newbot para começar a criação de um novo bot.
  3. Informe o nome visível do bot. Esse é o nome que as pessoas verão nas conversas.
  4. Escolha o nome de usuário público do bot. Ele precisa ser único e terminar com bot, como meu_atendimento_bot.
  5. Copie o token retornado pelo @BotFather e cole no campo Bot token da tela anterior.
  6. Volte para a configuração do Telegram, clique em Validar conexão e revise os dados preenchidos automaticamente.
Dica

Se quiser trocar foto, descrição ou comandos do bot depois, isso também pode ser feito pelo próprio @BotFather.
